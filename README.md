# Heather's Helper

Sequential task timer for Android — run through a routine one step at a time.

## What's in this repo

- **`lib/`** — Flutter Android app (timer screen, theme, branding)
- **`mockup/`** — Interactive Cursor canvas UI mockup (full app design: timer, tasks, alerts, settings)
- **`android/`** — Android build configuration and launcher icons

## Flutter app

```bash
flutter pub get
flutter run
```

Build a release APK:

```bash
flutter build apk
```

Requires Flutter SDK and Android SDK (`JAVA_HOME` configured for Gradle).
