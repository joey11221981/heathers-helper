# Heather's Helper — UI mockup

Interactive Cursor canvas mockup for the full app design (timer, tasks, alerts, settings).

## Open the mockup

1. Copy `heathers-helper-mockup.canvas.tsx` into your Cursor project's `canvases/` folder, or open it from this repo in Cursor.
2. Click **Open Canvas** on the file to run the live, clickable phone preview.

The mockup uses `useCanvasState` so tab choices, tasks, and preferences persist between sessions.
